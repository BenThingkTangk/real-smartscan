# R.E.A.L. SmartScan AI Engine 2.0

**Health Commerce Intelligence Platform**

The SmartScan AI Engine is the digital intelligence layer powering the Real Fresh / Real Mart ecosystem — a premium health & wellness superstore competing with Walmart at a 5-star level across the Caribbean and beyond.

## What It Does

- 🔍 **AI-Powered Search** — Search any product, get instant price comparison across Amazon, Walmart, Real Fresh, StockX, GOAT, Target, eBay, and 20+ sources
- 📷 **Barcode Camera Scanner** — Real `getUserMedia` + ZXing barcode decoding. Point your phone at any barcode in-store and see if it's a better deal elsewhere
- 🧠 **AI Deal Analysis** — 90-day price history analysis, deal score (1–100), BUY/WAIT/SKIP verdict per product
- 🔥 **Live Deal Feed** — Real-time price drops, filterable by category, with timer countdowns
- 🏪 **Real Fresh 20-Zone Ecosystem** — All 20 store zones with SKU counts, revenue, and margins
- 📺 **Real TV Live** — Live commerce channel integration (QVC meets Netflix for health)
- 📊 **Margin Engine** — Category take-rate sliders, hidden embedded margin breakdown, revenue model
- 💚 **Real Score™** — Personalized health intelligence dashboard
- 👑 **Plans** — Free / Prime Elite ($9.99/mo) / B2B API ($499/mo)

## Technology

- Single-file HTML/CSS/JS (no backend required — prototype with rich simulated data)
- ZXing.js for real barcode scanning via device camera
- Chart.js for 90-day price history sparklines
- Cabinet Grotesk + Satoshi fonts via Fontshare
- Dark/light mode toggle
